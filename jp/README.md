# lock_setting
Linux(GNOME)のシステム設定を固定するためのツールです。

## 使い方（有線ネットワークのロック）
ターミナルで以下のコマンドを実行してください。GUIが立ち上がり、ロックまたは解除を選択できます。
```bash
curl -s [https://raw.githubusercontent.com/appipinopi/lock_setting/main/jp/lock_network.sh](https://raw.githubusercontent.com/appipinopi/lock_setting/main/jp/lock_network.sh) | bash
